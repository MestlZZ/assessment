Nesting Elements Structure

class="<element-name>-container"
class="<element-name>-wrapper"
Element container (aka holder) - is usually used for placing element relatively to other elements.
Element wrapper - is usually used for centering (overflowing) element inside container.
CAUTION: usually developer can use styles for both container and wrapper in single container.

Variables naming

Correct

@main-width: 980px;
@main-color: #4d4d4d;
@main-margin: 30px;

Incorrect

@maincolor: #4d4d4d;
@mainWidth: 980px;
@MainMarGin: 30px;
In variables naming please use dash separation instead of case changing.
